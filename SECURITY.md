# Security Policy

## Supported Versions
This repository is research/evaluation software. Only the latest release tag is considered supported.

## Reporting a Vulnerability
If you believe you have found a security vulnerability:
- Do **not** post exploit details in a public issue.
- Open a GitHub issue titled: `SECURITY: <short description>`
- Include enough detail to reproduce safely.

This repo does not run network services by default. Most issues will be dependency or CI related.
